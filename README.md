External Pressure Analysis of Cylindrical AUV Hull using ANSYS
📌 Objective

To evaluate structural performance of a cylindrical pressure hull under 10 MPa external hydrostatic pressure.

🛠 Software Used

ANSYS Workbench

Static Structural Module

📐 Model Description

Cylindrical hull with hemispherical end caps

Structural Steel material

External pressure applied on outer surface

🔍 Boundary Conditions

External Pressure: 10 MPa

Fixed Support on one hemispherical end

📊 Results

Maximum Total Deformation: 0.263 mm (inward)

Maximum Equivalent Stress: 185.98 MPa

Yield Strength: 250 MPa

Factor of Safety ≈ 1.34

The structure remains below yield strength under applied pressure. Since external pressure vessels are primarily governed by buckling instability, further eigenvalue buckling analysis is recommended.
